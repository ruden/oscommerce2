Vanilla osCommerce
==================

Vanilla osCommerce this fork osCommerce v2.3.5

Compatibility
-------------

* PHP 7.X or PHP 8.X
* MySQL 5.7.8+ or MariaDB 10.2.7+
* Apache 2.2 and 2.4

Demo: <http://demo.osclab.com/vanilla-oscommerce/>

Download: <https://github.com/ruden/vanilla-oscommerce/archive/dev.zip>
